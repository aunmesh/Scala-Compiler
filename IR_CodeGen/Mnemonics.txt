       Three Address Code Format

|---------------------+--------------------|
| INSTRUCTION         | Three-address code |
|---------------------+--------------------|
| Assignment          |                    |
|---------------------+--------------------|
| x = y               | = x y              |
|---------------------+--------------------|
| Operation           |                    |
|---------------------+--------------------|
| x = op y            | op x y             |
| x = y op z          | op x y z           |
|---------------------+--------------------|
| Function            |                    |
|---------------------+--------------------|
| call f              | call f             |
| ret v               | ret v              |
|---------------------+--------------------|
| Jump                |                    |
|---------------------+--------------------|
| goto L              | goto L             |
| if x relop y goto L | ifgoto L relop x y |
|---------------------+--------------------|
| Pointer             |                    |
|---------------------+--------------------|
| x = &y              | = x & y            |
| x = *y              | = x * y            |
| *x = y              | = * x y            |
|---------------------+--------------------|
| Indexed Assignment  |                    |
|---------------------+--------------------|
| x = y[i]            | = x y [ i ]        |
| x[i]=y              | = x [ i ] y        |
|---------------------+--------------------|
| Library functions   |                    |
|---------------------+--------------------|
| scan a              | scan a             |
| print a             | print a            |
| print "\n"          | print newline      |
|---------------------+--------------------|
| Array Declaration   |                    |
|---------------------+--------------------|
| a = Array[Int] (n)  | Array a 4*n        |
|---------------------+--------------------|

Note: Every three-address code is prefixed with a corresponding line number[not shown for clarity].
